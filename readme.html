Tentang Aplikasi
      Aplikasi Jadwal merupakan sebuah aplikasi sistem informasi sekolah berbasis web. Aplikasi ini dapat menyimpan data-data yang ada di Sekolah seperti: Data Guru, Data Siswa, Data Kehadiran, Data jadwal, dan lain sebagainya. Aplikasi ini dibuat agar memudahkan pekerjaan Guru dan operator Sekolah dalam memanajemen tugas-tugasnya.
